Getting Started

×I assume you have your own wordpress installion, so let's get started!

Installion

You can install the theme in two ways:

  1. FTP Upload: Unzip the “avalance-theme.zip” file and upload the contents to the /wp-content/themes folder on your server.
  2. Wordpress Upload: Navigate to Appearance > Install Themes > Upload. Select the zipped theme folder. Hit Install Now and the theme will be uploaded and installed.

When your theme has been installed successfully, hit Activate to get it work.


Activate Theme Options

When your newly uploaded theme has been activated, you will notice that a new submenu item Theme Options appeared under the Appearance menu. Open it, then hit Save Changes.



Activate Plugins

At the top of your admin, you see a large yellow notice to activate AvaLance's required plugins.

Hit Begin Install and when it is successfully installed, navigate to Plugins > Installed Plugins and activate the newly installed AvaLance Portfolio Post Type plugin.



Import Demo Content

If you would like to have the same content like in AvaLance's live preview, you need to import demo content. To do that, navigate to Tools > Import > Wordpress, then select Extras/Import/avalance-demo-content.xml from the main theme folder. On the next page, check Download and import file attachments option and hit Submit.

If your import was successfull, go to Appearance > Menus and in the Menu settings select Top primary menu and hit Save Menu.

Finally go to Appearance > Options Backup and copy paste the content of avalance-theme-options.txt from the Extras/Import folder in the download then click Import Theme Options.


Note: The photos from the live preview are shown as grey blank images, they not included with the download as they are copyrighted files from ShutterStock.

Set Up Home Page

  • First go to Pages > Add New and create a basic page called Home or anything you like, and select page template Home on the right.
  • Go to Settings > Reading and in the Front Page Displays select Static Page then select your newly created page Home

Next go to Settings > Permalinks and select Post Name and hit Save Changes


Note: The photos from the live preview are shown as grey blank images, they not included with the download as they are copyrighted files from ShutterStock.

Theme Options

×Navigate to Appearance > Theme Options to set a lot of cool stuff for the theme and edit parts of the main content.

General Options

Here you can upload your logo and favicon for your site, and set the color scheme and the 404 text what will show up on 404 error pages.



Set Up Slider

First you need to enable the slider, go to Appearance > Theme Options > Slider and check the Display Slider. Below you can select a pattern overlay for the slider images if you want.

To add slides, click the Add New button below Slides.




Create your slides using the picture above as a reference. You can add as many slides as you want, but it is recommended to keep under 3-4 slides to avoid longer page loading. When you are ready with your slides, click Save Changes and now you can see your slides by visiting the home page.

If you don't want a slider just a static image as a background, just upload one slide.

Note: For the best result, please upload images for slides in a 2000x1330px size.

Set Up Header

Go to Appearance > Theme Options > Header and you can do a lot of cool stuff here to customize your header.

If you don't need one of the features (e.g. header logo) just leave it empty.


After filling the fields with your needed contents just hit Save changes .




Set Up Our Team

  1. First you need to create a new page at Pages > Add New and set the page Template to Our Team. The title will show up as the section's title, and you can add content if you wish.
  2. Next to Appearance > Theme Options > Our Team and click Add New then fill the icon box with your details. It is optional to use with 4 team members.

      3. You can use the options below as a reference. If you're ready with adding your members, just hit Save Changes and for now you'll see the team members on your site.



Set Up Our Process

  1. First you need to create a new page at Pages > Add New and set the page Template to Our Process. The title will show up as the section's title, and you can add content if you wish.
  2. Next go to Appearance > Theme Options > Our Process, upload a background image for this section, then click Add New to create new process part and fill the icon box with your details. It is optional to use 4 process parts.

For the description image it's optional to upload a 156x156px image.

The custom flat icon are included with the theme in the additional color schemes, feel free to use them in your design too. You can find them in the Extras/Icons folder in the main download.






Set Up Services & Testimonials

  1. First you need to create a new page at Pages > Add New and set the page Template to Our Services. The title will show up as the section's title, and you can add content if you wish.
  2. Next go to Appearance > Theme Options > Our Services and click Add New below the Service Boxes or the Testimonials then fill the icon box with your details. For service boxes it's optional to use 3 of them, while testimonials you can add as many as you want.





Set Up Contact & Offices

  1. First you need to create a new page at Pages > Add New and set the page Template to Contact. The title will show up as the section's title, and you can add content if you wish.
  2. Next go to Appearance > Theme Options > Contact and add the hit Add New beside the Contact Details section. It's optional to use three of them regarding the layout. After saving, they can be reordered with drag and drop features.



      3. If you need the offices layout too, just fill the Offices details below Contact Details and your map and details will appear under the contact details. If you need just one map, leave blank the other fields.

Note: For the best result, please upload images for the maps in 460x305px size.


Set Up Footer

  1. Go to Appearance > Theme Options > Footer and add your copyright text and social network links then finally hit Save Changes



Set Up Portfolio

×There is a custom portfolio post type included with AvaLance, you can find it in the main navigation after activating the plugin.

Create Portfolio Section

First you need to create a new page at Pages > Add New and set the page Template to Portfolio. The title will show up as the section's title, and you can add content if you wish.





Creating Filters

Go to Portfolio > Filters and enter a name, slug (URL friendly name) for the filter, and in the Description field add your icon's name ( icon reference ), and finally hit Add New Filter.





Adding Portfolio Items

After activating the custom portfolio post type plugin, go to Portfolio > Add New and enter a title for the project and upload a featured image (480x296px is optional), then set a filter for it from the newly created filters, add tags if you wish and hit Publish.





Portfolio Details Page

For every portfolio item you can create a custom portfolio details page by adding content to the portfolio item's editor. That content will show up when the portfolio item thumbnail is clicked. You can add any content you wish or included shortcodes like clients shortcode etc.

Creating Tablet Slider
  1. Change to Text editor and click Add Media button
  2. In the pop up window select Create Gallery and select your uploaded images for the tablet slider (720x514px optional)
  3. Hit Create New Gallery and then Insert Gallery
  4. In the gallery's shortcode, add this: size="full"

Finally your gallery's shortcode should look something like this: [gallery columns="1" link="none" size="full" ids="106,107,108"]




Content Editing

×Beside custom posts and sections on the home page, you can create individual pages for your contents.

Creating Pages

To create an individual page navigate to Pages > Add New and set the page Template to Page. Add the title for the page, add content for it (i will get on the available shortcodes in the next section) and click Publish to create your new page.

Note: If you would like to add your page in the main navigation, you have to set the menu item to Custom and add the link manually. Read more about this in the Menus section.

Shortcodes

AvaLance has some shortcodes to make content editing easier. In the following you can see the available shortcodes with a short explanation and a code sample what you can just paste in your editor in Text view.


Columns Shortcode

AvaLance is based on the Skeleton Grid system, it has 16 columns by default. You can easily add columns to your content, just take a look at the code sample of the usuage.

It is recommended to put everything in these columns if you are editing your own content, for example an individual page.

[column width="1/2"]This is the first half column[/column]
[column width="1/2"]This is the second half column[/column]

Width options:

  • 1, full
  • 1/2, eight
  • 1/3, one-third
  • 1/4, one-fourth
  • five, six, seven etc.


Lead Shortcode

The lead shortcode is the center-aligned introduction, what can be seen under the section titles in my demo.

[lead]Lorem ipsum dolor sit amet consescetur elit adipiscin massa[/lead]


Divider Shortcode

The divider shortcode is a 1px full width solid grey divider, it can be used for separating different content sections.

[divider]


Titles Shortcode

You can define titles for your content from a variety of h1-h6 tags. You can see the theme demo's Shortcodes section for reference of the headlines layout.

[title type="h2"]This is a h2 headline[/title]

Type Options:

  • h1, h2, h3, h4, h5, h6


Button Shortcode

You can easily add buttons with this shortcode, just set the size and URL for the button.

If you leave every settings blank (except the src of course), by default the button is medium sized.

[button size="large" src="http://www.link.com"]Click Me[/button]

Button Options:

  • Size: small, medium, large
  • Color: colored, light, dark
  • Src: your url

Menus

AvaLance is using WP 3.0 custom menus, to create your own navigate to Appearance > Menus and click create new menu. Add any title to your menu you want (e.g.: custom menu) and add your existing pages from the left, and set the menu settings below the menu to Top Primary Menu and hit Save Menu.
